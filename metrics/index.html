Syllabus

These notes provide a detailed introduction to statistical and econometric theory. These notes are useful for anyone who wishes to do serious quantitative/statistical research in the social sciences.

  • Note: This only covers the theory of statistics/econometrics. If you want to implement these methods for research, you will need to consult the other notes.

This course is also a pre-requisite for the subsequent applied econometrics/statistics notes:

  • Applied Econometrics Methods
  • Multivariate Statistical Methods
  • Time Series Analysis
  • Applied Machine Learning

Prerequisites: Strong skills in algebra and single variable calculus. Basics of statistics (what a random variable is, what probability is, some ideas behind distributions). Basic linear algebra is helpful.

  • You can undertake this course without these skills, but you may struggle a little.


Part I: Basics of Statistical Theory

This part provides the basics of statistical concepts, theory, and discusses statistical inference.

  1. Introduction and Basics of Statistics
  2. Estimates and Statistical Inference
  3. Relationships with Correlation and Covariance


Part II: Linear Regression and OLS

This part explores the most commonly used estimator: the Ordinary Least Squares (OLS) Estimator, and how it can be useful for estimating relationships between variables.

  1. Simple Linear Regression: Basics and Estimation
  2. Simple Linear Regression: Properties and Inference
  3. Multiple Linear Regression: Basics and Estimation
  4. Multiple Linear Regression: Properties and Inference
  5. Asymptotics and Homoscedasticity
  6. Extensions of Linear Regression


Part III: Further Estimators and Models

Sometimes, OLS is not a good estimator. This part explores other estimators and models, and how they can be used to solve some of the issues with OLS and linear regression.

  1. Method of Moments Estimator
  2. Instrumental Variables Estimator
  3. Logistic Regression and the Maximum Likelihood Estimator
  4. Multinomial and Ordinal Logistic Regression
  5. Regression for Count Outcomes


What to Learn Next

The following notes on applied econometrics/statistics will provide the necessary skills to implement the theory into actual research.
